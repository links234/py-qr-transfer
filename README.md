# QR Transfer
Python script for file transfer over QR for [Bug Koops][1] android application.

# Dependencies
You need to have both Python2.x and Python3.x installed with following packages installed:

 - qrcode, numpy, PIL and six (pip3)
 - wx and zlib (pip)
